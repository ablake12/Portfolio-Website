# Portfolio-Website

## To run locally:
1. Open the terminal and navigate to the website directory:
- ```cd website```
2. Export the environment variable ```FLASK_APP```:
- export FLASK_APP=website.py
3. Run locally:
- ```flask run```
To debug:
- ```flask run --debugger```
For other users:
- ```flask run --host=0.0.0.0```
4. Go to the given port and enjoy!
